## 🎯 프로젝트 개요

### 핵심 컨셉
마케터와 콘텐츠 크리에이터를 위한 **실시간 밈/유행 트렌드 백과사전 + 이모티콘 생성 도구**

### 핵심 가치 제안
"트렌드를 놓치지 않고, 실전에서 바로 쓸 수 있는 밈 백과사전 + 이모티콘 생성 도구"

### 프로젝트 비전
- 마케터가 가장 먼저 찾는 트렌드 레퍼런스 플랫폼
- 밈을 단순히 이해하는 것을 넘어 실전 마케팅에 활용하는 도구
- 커뮤니티와 AI가 결합된 하이브리드 콘텐츠 생태계

## 🔥 문제 정의

### 배경

디지털 마케팅 환경에서 밈과 인터넷 트렌드는 브랜드 커뮤니케이션의 핵심 도구로 자리잡았다. 그러나 다음과 같은 문제들이 존재한다:

### 핵심 문제

| # | 문제 | 영향 |
|---|------|------|
| P1 | **밈의 빠른 생명주기** — 밈은 수일~수주 단위로 유행과 소멸을 반복하며, 마케터가 수동으로 추적하기 어렵다 | 트렌드 활용 타이밍을 놓쳐 마케팅 효과 저하 |
| P2 | **브랜드 안전성 판단의 어려움** — 밈의 맥락과 이면의 뉘앙스를 파악하지 못해 부적절한 밈 사용으로 브랜드 이미지가 손상될 수 있다 | 브랜드 리스크, 소비자 반발, PR 위기 |
| P3 | **세대 간 언어 격차** — MZ세대/Gen Alpha의 밈 언어를 기성 마케터가 이해하기 어렵다 | 타겟 세대와의 소통 단절, 억지 밈 사용으로 역효과 |
| P4 | **산재된 정보** — 밈 관련 정보가 SNS, 커뮤니티, 유튜브 등에 분산되어 체계적 관리가 불가능하다 | 정보 수집에 과도한 시간 소요, 일관성 부재 |
| P5 | **마케팅 활용 가이드 부재** — 밈의 의미는 알아도 어떻게 브랜드에 적용할지 가이드가 없다 | 밈 활용도 저하, 잘못된 활용으로 인한 역효과 |

### 기회

- 국내에 마케터 특화 밈 사전 서비스가 부재
- 밈 마케팅 시장 성장 (브랜드의 SNS 밈 활용 증가 추세)
- AI 기술을 활용한 카피 생성, 이모티콘 제작 등 확장 가능성

## 💡 솔루션

### AI-Native 해결책

1. **실시간 밈 트렌드 탐지**
   - 인스타 릴스, 스레드, X(트위터), 틱톡에서 급상승 키워드 자동 수집
   - AI가 맥락 분석해서 "이 밈은 왜 유행하는지" 3줄 설명
   - 예시: "~하다못해" → "극단적인 상황을 표현할 때, 자조적 유머로 사용"

2. **마케터용 AI 사용 가이드**
   - 단순 의미 설명이 아니라 "어떤 브랜드가 어떻게 써야 하는지" 제안
   - "이 밈은 Z세대 타겟 캐주얼 브랜드에 적합 / B2B는 피하세요"
   - 예시 카피 자동 생성: "우리 제품으로 ~하다못해 성공 스토리"

3. **AI 이미지 생성으로 밈 이모티콘 제작**
   - 텍스트 프롬프트 or 사진 업로드 → AI가 이모티콘 생성
   - 커뮤니티 제작 이모티콘 큐레이션
   - 마케팅 콘텐츠에 바로 활용 가능한 PNG/SVG/GIF 제공

4. **플랫폼별/세대별 트렌드 분리**
   - 인스타에서만 유행 vs 전 플랫폼 확산 중 vs 이미 쇠퇴기
   - 2030 vs 4050 반응 차이 분석
   - "이 밈은 틱톡에선 유행하지만 링크드인에선 역효과"

5. **주간 트렌드 리포트**
   - 매주 월요일 "이번 주 꼭 알아야 할 밈 TOP 5" 메일 발송
   - "저번 주 가장 많이 검색된 밈" 랭킹
   - "내가 저장한 밈" 북마크 기능

### 3️⃣ 밈 이모티콘 스튜디오 🎨

#### A. 이모티콘 제작 방식

**① AI 이미지 생성**

```
사용자 입력:
- 텍스트 프롬프트: "제곱근 하트를 귀여운 캐릭터로 만들어줘"
- 참고 이미지 업로드: [밈 원본 이미지]
- 스타일 선택: 귀여움/심플/미니멀/레트로
  ↓
AI 이미지 생성
  ↓
4개 옵션 제시 → 사용자 선택 → 다운로드
```

**② 커뮤니티 제작 + 큐레이션**

```
디자이너/일반 사용자:
  ↓
이모티콘 제작 후 업로드
  ↓
관리자/AI 검증 (저작권, 퀄리티)
  ↓
승인 → 밈 상세 페이지에 등록
  ↓
인기 이모티콘 TOP 10 큐레이션
```

#### B. 제공 포맷 (다중 다운로드)

- **PNG** (투명 배경, 512x512px) - SNS 콘텐츠용

### 기존 서비스 vs 밈 사전

| 항목 | 기존 신조어 사전 | 밈 사전 |
|------|------------------|---------|
| **콘텐츠 범위** | 신조어 의미만 설명 | 유행어/템/음식/챌린지/음악 모두 |
| **정보 깊이** | 단순 의미 정의 | 마케팅 실전 가이드 + 카피 예시 |
| **실시간성** | 정적 정보 | 실시간 트렌드 점수 + 생명주기 |
| **활용 도구** | 텍스트만 제공 | 바로 쓸 수 있는 이모티콘 |
| **타겟** | 일반 사용자 | 마케터 특화 |
| **원본 출처** | 없음 | 유튜브/인스타/틱톡 원본 링크 |
| **AI 기능** | 없음 | 카피 생성 + 이모티콘 생성 |
| **재미 요소** | 없음 | 밈고사 퀴즈 |
| **커뮤니티** | 없음 | 이모티콘 제작/공유 |

### 핵심 경쟁력

1. **마케터 특화** - 단순 의미가 아닌 "어떻게 써야 하는지" 실전 가이드
2. **AI 통합** - 카피 생성 + 이모티콘 생성 + 조합 추천
3. **실시간 트렌드** - 생명주기 그래프 + 트렌드 점수
4. **즉시 활용 가능** - PNG/SVG/GIF 다운로드, 커스터마이징
5. **커뮤니티 하이브리드** - AI 생성 + 커뮤니티 제작 결합
6. **벤치마킹 데이터** - 경쟁 브랜드 활용 사례 제공
7. **재미 + 학습** - 밈고사로 게이미피케이션
