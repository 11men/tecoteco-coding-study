3️⃣ 유행 카테고리 확장

🗂️ 카테고리 세분화

A. 유행어 (기존)

단어/표현 (예: "~하다못해", "제곱근 하트")
신조어 (예: "갓생", "워라밸")
B. 유행템 ✨ (신규)

예시: "스탠리 텀블러" (2024)
- 카테고리: 패션/라이프스타일
- 유행 이유: "하이드로 플라스크 이후 물병 열풍"
- 마케팅 활용: "우리 브랜드 X 스탠리 콜라보"
- 타겟: 2030 여성
- 플랫폼: 인스타그램, 틱톡
C. 유행 음식 🍜 (신규)

예시: "마라탕후루" (2024)
- 카테고리: 식품
- 유행 이유: "마라탕 + 탕후루 조합의 충격"
- 마케팅 활용: "신메뉴 네이밍 벤치마킹"
- 타겟: 전 연령
- 플랫폼: 틱톡, 인스타 릴스
D. 유행 챌린지 🎪 (신규)

예시: "Random Play Dance"
- 카테고리: 챌린지
- 유행 이유: "K-POP 팬덤 문화"
- 마케팅 활용: "브랜드 자체 챌린지 기획 시 참고"
- 타겟: 1020
- 플랫폼: 틱톡, 유튜브 쇼츠
E. 유행 음악 🎵 (신규)

예시: "APT. (ROSÉ & Bruno Mars)"
- 카테고리: 음악
- 유행 이유: "릴스/쇼츠 배경음악으로 바이럴"
- 마케팅 활용: "브랜드 영상 BGM 추천"
- 타겟: 전 연령
- 플랫폼: 인스타 릴스, 틱톡

4개 옵션 제시 → 사용자 선택 → 다운로드
디자이너/일반 사용자:
  ↓
이모티콘 제작 후 업로드
  ↓
관리자/AI 검증 (저작권, 퀄리티)
  ↓
승인 → 밈 상세 페이지에 등록
  ↓
인기 이모티콘 TOP 10 큐레이션
기본 이모티콘 선택
  ↓
커스터마이징 옵션:
- 색상 변경 (브랜드 CI 적용)
- 텍스트 추가 ("SALE", "NEW" 등)
- 표정/크기 조절
- 배경 추가/제거
  ↓
실시간 프리뷰 → 다운로드
밈 1 선택: "제곱근 하트"
밈 2 선택: "워라밸"
  ↓
AI가 조합 이모티콘 자동 생성
  ↓
결과: "√(워라밸) 달성!" 이모티콘
각 이모티콘마다 라이선스 표시:
✅ 상업적 사용 가능 (CC0)
⚠️ 출처 표기 필수 (CC BY)
❌ 개인용만 가능 (Personal Use)
💰 유료 라이선스 (저작권자 연락 필요)
입력:
- 선택한 밈: "제곱근 하트"
- 브랜드 정보: "친환경 화장품 브랜드"
- 캠페인 목적: "발렌타인데이 프로모션"
  ↓
AI 출력:
1. "√(자연)에서 온 사랑의 선물"
2. "√(순수)한 마음을 담았습니다"
3. "고객님께 √(감사)를 전합니다"
현재 시점: 11월
  ↓
자동 추천:
- 블랙프라이데이 활용 가능 밈 5개
- 크리스마스 시즌 밈 8개
- 연말 결산용 밈 3개
[⚠️ 주의] "이 밈은 최근 논란이 있었습니다"
- 논란 사유: "특정 집단 비하 논란"
- 발생 일시: 2024.10.15
- 대체 밈 추천: [안전한 대체재 3개]
내 산업 설정: "패션 브랜드"
  ↓
대시보드 자동 큐레이션:
- 패션 브랜드 추천 밈
- 패션 업계에서 실제 활용한 사례
- 패션 타겟(2030 여성)에게 통하는 밈
┌─────────────────────────────────────┐
│ 🔥 오늘의 HOT 밈 TOP 10              │
│ 1. √하트 (↗ 트렌드 점수 95)         │
│ 2. ~하다못해 (↘ 트렌드 점수 88)     │
│ ...                                  │
├─────────────────────────────────────┤
│ 📱 플랫폼별 트렌드                   │
│ [인스타] [틱톡] [X] [유튜브]        │
├─────────────────────────────────────┤
│ 🎯 내 산업 추천 밈 (패션)            │
│ - 밈 A (적합도 ⭐⭐⭐⭐⭐)             │
│ - 밈 B (적합도 ⭐⭐⭐⭐)              │
├─────────────────────────────────────┤
│ 📅 이번 주 시즌 밈 (크리스마스)      │
│ - 시즌 밈 5개 추천                   │
└─────────────────────────────────────┘

크롤링:
Puppeteer/Playwright (웹 크롤링)
Instagram API / TikTok API (공식 API)
임베드:
YouTube oEmbed API
Instagram Embed API
TikTok oEmbed API
Infra

Hosting: Vercel (Frontend) + AWS/Railway (Backend)
CI/CD: GitHub Actions
Monitoring: Sentry (에러), Posthog (분석)
