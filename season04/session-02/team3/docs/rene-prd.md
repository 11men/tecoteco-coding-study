## 서비스 핵심 컨셉

**Data Mirroring (데이터 미러링)**

미래 예측이 아닌 **과거 대조**를 통해, FOMO 순간에 "지금 사면 어떻게 되는지"를 데이터로 보여주는 서비스.

- 리딩방: "지금 당장 사!" → 욕망의 배설구
- Parallel Investor: "과거에 똑같이 한 사람은 망했어" → **욕망의 필터**

**핵심 차별점**: 모든 투자 앱이 "사게 만들" 때, 유일하게 **"사지 말라"고 데이터로 말하는** 서비스.

## 피처 A. Pattern Similarity Engine (패턴 유사도 매칭)

### 분류: 핵심 엔진 (Core)

### 한 줄 요약
사용자가 "사고 싶다" 순간, 과거 10년치 데이터 중 유사 패턴을 찾아 결과를 보여준다.

### 작동 흐름

| 단계 | 기능 | 설명 |
|------|------|------|
| 1 | Moment Capture | "사고 싶다" 클릭 → 현재 차트 패턴(이동평균선, RSI, 거래량) 캡처 |
| 2 | Historical Sync | 과거 10년치 데이터 중 유사도 80%+ 구간을 AI가 탐색 |
| 3 | Parallel Result | "이 패턴에서 산 사람: -25%. 15일 뒤 진입한 사람: +18%" 제시 |
| 4 | Action Guide | "지금 사시겠습니까? 2주 후 타점을 예약하시겠습니까?" |

### 사용자 가치
- FOMO 순간에 **객관적 판단 근거** 제공
- "예측"이 아닌 "과거 팩트" → 틀릴 수 없음

### 기술 요소
- 차트 패턴 분석: 이동평균선, RSI, 거래량
- 유사도 매칭: 코사인 유사도 (MVP) → 향후 AI 고도화
- AI 엔진: Claude API 활용 (패턴 분석 + 결과 메시지 생성)

### MVP 범위
- 종목 검색 + 현재 가격 자동 반영
- 과거 유사 패턴 매칭 결과 카드
- "사면 / 참으면" 시나리오 비교 UI

## 피처 B. Shadow Record (참음의 기록)

### 분류: 핵심 기능 (Core)

### 한 줄 요약
FOMO를 느꼈지만 참은 종목을 기록하고, 시간 경과 후 결과를 알려준다.

### 기록 항목
- 종목명
- 당시 가격 (자동 캡처)
- 투자하려던 금액
- FOMO 강도 (1~5 스케일)
- 참은 이유 (메모)

### 결과 피드백

| 상황 | 피드백 |
|------|--------|
| 하락 시 | "축하합니다! 참으신 덕분에 50만원을 지켰습니다. (방어 성공)" |
| 상승 시 | 알림 없음 / "시장 수업료를 아꼈다고 생각하세요" (설정 가능) |

### 사용자 가치
- 충동 매수를 **기록 행위**로 전환
- 방어 성공 시 **구체적 보상감**
- "참는 것 = 돈 버는 것"을 뇌가 학습

### MVP 범위
- FOMO 기록 폼 (종목, 금액, FOMO 강도, 메모)
- 1주 / 1개월 후 결과 알림

## 피처 C. JOMO Calculator (자산 방어 시각화)

### 분류: 보상 기능 (Reward)

### 한 줄 요약
방어한 손실액을 실물 가치로 환산하여 구체적 보상감을 준다.

### 환산 예시

| 미매수 종목 | 투자 예정액 | 하락률 | 방어 금액 | 실물 환산 |
|------------|-----------|--------|----------|----------|
| A코인 | 100만원 | -20% | 20만원 | 치킨 10마리 |
| B주식 | 500만원 | -35% | 175만원 | 최신형 맥북 1대 |

### 사용자 가치
- 추상적 금액 → **구체적 실물**로 체감 극대화
- 누적되면 "참는 것 = 돈 버는 것" 학습 강화
- SNS 공유 요소 (바이럴)

### MVP 범위
- 방어 금액 실물 환산 카드

## 피처 D. Patience Leveling (참을성 레벨링)

### 분류: 게이미피케이션 (Engagement)

### 한 줄 요약
참음 횟수와 기간에 따라 레벨과 뱃지를 부여하는 성장 시스템.

### 레벨 체계

| 레벨 | 칭호 | 영문명 | 조건 |
|------|------|--------|------|
| Lv.1 | 펄럭귀 | Paper Ear | FOMO 기록 시작 |
| Lv.5 | 철벽 | Iron Wall | 5회 참음 성공 |
| Lv.10 | 돌부처 | Stone Buddha | 10회 참음 + 폭락장 생존 |
| Lv.30 | 현금 수호자 | Cash Guardian | 30일 연속 미매매 |
| Lv.99 | 버핏의 후계자 | Heir of Buffett | 전설의 참을성 달성 |

### 특수 뱃지

| 뱃지 | 조건 |
|------|------|
| 생존자 | 폭락장(-10%+)에서 매매 0회 |
| 빙쇼트 | 대세 상승장에서 추격매수 없이 버팀 |
| 해탈 불가 | 미매수 기록한 종목이 실제 상폐 |

### 사용자 가치
- "안 하는 것"에 **성취감과 정체성** 부여
- 연속 기록 유지 동기 (손실 회피의 긍정적 활용)
- 수집 욕구 자극

## 피처 E. Re-Entry Signal (역발상 타점 알림) - Premium

### 분류: 유료 기능 (Monetization)

### 한 줄 요약
참은 뒤, 과거 패턴상 "진짜 반등이 시작된 골든 타임"에 유료 알림을 보낸다.

### 설계 의도
- 참으라고만 하면 사용자는 떠남
- 사용자의 본질적 욕구는 **"돈을 버는 것"**
- 참은 후 최적 진입 타점까지 알려주면 지불 의향 극대화

### 수익 모델 연결
- Free: 기본 FOMO 기록 + 1주일 후 결과 + 기본 방어 금액 + 레벨링
- **Premium (월 9,900원)**: 실시간 패턴 분석 + 상세 시뮬레이션 + AI 코칭 + Re-Entry Signal

---

## 피처 F. 충동 패턴 분석 리포트 - Premium

### 분류: 유료 기능 (Insight)

### 한 줄 요약
사용자의 FOMO 기록을 분석해서 "언제, 어떤 상황에서 충동이 심한지" 패턴을 보여준다.

### 예시 인사이트
- "나는 월요일 아침에 FOMO가 심하다"
- "뉴스를 보고 기록한 경우 하락 확률 85%"
- "FOMO 강도 4 이상일 때 참음 성공률이 오히려 높다"

### 사용자 가치
- **메타인지**: 내 충동 패턴을 객관적으로 인식
- 자기 이해 → 자기 통제로 이어지는 선순환

## 사용자 플로우 (전체 여정)

```
1. FOMO 발생
   SNS/커뮤니티에서 "이거 사야 하나?" 충동
       │
       ▼
2. 앱 진입
   매수 버튼 대신 Parallel Investor에 종목/금액 입력
       │
       ▼
3. Pattern Matching [피처 A]
   AI가 과거 유사 패턴 분석 결과를 실시간 제시
   "이 패턴에서 산 사람: -25%"
       │
       ▼
4. Action Decision
   데이터를 보고 "산다 / 참는다" 결정
       │
       ├─ 참는다 ──→ 5. Shadow Record [피처 B]
       │              FOMO 강도와 함께 기록
       │                    │
       │                    ▼
       │              6. 결과 피드백 (1주/1개월 후)
       │              하락 → "방어 성공!" [피처 C: JOMO 환산]
       │              상승 → 위로 메시지
       │                    │
       │                    ▼
       │              7. 성장 확인 [피처 D]
       │              누적 방어 금액 + 레벨 + 배지
       │
       └─ 산다 ──→ 결과 추적 → 충동 패턴 분석 [피처 F]
```

| Phase | 영역 | 설명 |
|-------|------|------|
| 1 | 투자 FOMO (현재) | 주식/코인/ETF 충동 매수 방어 |
