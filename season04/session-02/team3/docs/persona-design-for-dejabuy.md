## 1. 페르소나란 무엇인가

### 정의

매수 충동이 터지는 **바로 그 순간**, 3초 질문으로 투자 성향을 태깅하는 기능.

당신이 왜 사려는지, 얼마나 들고 있을 건지, 얼마나 확신하는지를 **3글자 코드**로 기록한다.

그리고 **장 마감 후**, 데이터가 당신에게 말한다.

> "오늘도 뉴스 보고 샀네. 너는 뉴스형일 때 10번 중 9번 진다."

### 왜 필요한가

DÉJÀ BUY의 핵심은 **"또 사려고? 이거 전에도 봤잖아"** 다.

차트 패턴은 반복된다. **당신의 매수 패턴도 반복된다.**

문제는 본인이 모른다는 것이다.

- "나는 분석적 투자자야" → 데이터: 10번 중 8번이 뉴스 보고 충동매수
- "나는 장기투자자야" → 데이터: 평균 보유 기간 3일
- "나는 신중해" → 데이터: 확신 HIGH로 올인한 매수가 7번

**페르소나는 거울이다.** 매수 버튼을 누르는 순간의 당신을 기록하고, 나중에 그대로 보여준다.

왜곡 불가. 합리화 불가. **그때 그 순간, 당신은 그랬다.**

## 2. DÉJÀ BUY 구조 속 페르소나의 위치

### 현재 DÉJÀ BUY 구조 (3탭)

```
┌─────────────────────────────────────────────┐
│   🏠 홈         📝 기록         🏆 내 레벨    │
└─────────────────────────────────────────────┘

🏠 홈 탭: 패턴 분석
  → 실시간 차트 입력
  → 과거 패턴 매칭 (Déjà Buy 감지)
  → "당시 FOMO 매수자 73%가 2주 내 -23%"
  → [참는다 / 그래도 산다]

📝 기록 탭: Shadow Portfolio + JOMO
  → 참음 기록 목록
  → JOMO 계산기 (안 사서 번 돈)

🏆 내 레벨 탭: 레벨링 + 뱃지
  → Patience Leveling
  → 뱃지 컬렉션
```

### 페르소나가 들어갈 위치

페르소나는 **"그래도 산다" 버튼을 눌렀을 때** 작동한다.

```
[홈 탭] 패턴 매칭 후 선택지
        │
        ├─ [참는다] ─────→ [기록 탭] Shadow Portfolio
        │                  "참음 기록 완료. 7일 후 결과 알려드림."
        │
        └─ [그래도 산다] ──→ 🆕 [페르소나 3초 태깅]
                            "왜 사? 얼마나? 확신?"
                            ↓
                         [기록 탭] 매수 기록 (태그 포함)
                            ↓
                         장 마감 후 [회고 푸시]
                            ↓
                         [내 레벨 탭] 투자 MBTI 누적 통계
```

**핵심 원칙**:
- 페르소나는 **별도 탭이 아니다**.
- 기존 플로우에 자연스럽게 녹인다.
- "그래도 산다"를 누르면 → 자동으로 3초 태깅 시작 → 기록 탭에 쌓임 → 내 레벨 탭에서 통계 확인

## 3. 페르소나 UX 흐름 (DÉJÀ BUY 톤)

### Step 1: 홈 탭 - "그래도 산다" 버튼

사용자가 차트를 입력하고, 패턴 매칭을 확인한 후:

```
┌────────────────────────────────────────────┐
│  Déjà Buy 감지.                            │
│  현재 패턴은 2024.03 비트코인 급등과 89% 유사 │
│                                            │
│  당시 FOMO 매수자 73%가 2주 내 -23%.       │
│  3일 기다린 사람은 +15%.                   │
│                                            │
│  이 결말을 알고도 사시겠습니까?             │
│                                            │
│  [ 참는다 ]        [ 그래도 산다 ]         │
└────────────────────────────────────────────┘
```

**[그래도 산다]** 버튼 클릭 시 → 바로 Step 2로 이동

---

### Step 2: 3초 태깅 (한 화면, 3개 질문)

**디자인 원칙**:
- 한 화면에 3개 질문 모두 표시
- 각 질문마다 2-3개 선택지
- 버튼 터치 3번이면 끝
- 텍스트 입력 없음. 오직 선택.

```
┌────────────────────────────────────────────┐
│  매수 기록                                  │
│  (3초만 대답해 주세요)                      │
│                                            │
│  Q1. 왜 사려고?                            │
│  [ 뉴스/찌라시 ]  [ 차트 ]  [ 그냥 감 ]    │
│      (N)           (C)        (F)         │
│                                            │
│  Q2. 얼마나 들고 있을 건데?                 │
│  [ 오늘 단타 ]  [ 스윙 ]  [ 장기 존버 ]    │
│      (S)          (W)        (H)          │
│                                            │
│  Q3. 확신은?                               │
│  [ 인생 건다 ]       [ 정찰병만 ]          │
│      (H)               (L)                │
│                                            │
│  비트코인  65,000,000원                     │
│  100만원어치                                │
│                                            │
│            [ 기록 완료 ]                    │
└────────────────────────────────────────────┘
```

**UX 포인트**:
- 질문 텍스트는 **구어체**. "왜 사려고?" / "얼마나 들고 있을 건데?"
- 선택지는 **직관적 + 솔직한 표현**. "인생 건다" / "그냥 감"
- 코드 (N, C, F)는 작게 표시. 사용자에게 코드가 중요한 게 아니라 **의미**가 중요.

---

### Step 3: 결과 화면 - "당신의 매수 유형"

```
┌────────────────────────────────────────────┐
│  매수 기록 완료                             │
│                                            │
│  오늘 당신은:                               │
│  🔥 NFH - 불나방                           │
│                                            │
│  뉴스 보고(N) / 단타(S) / 인생 건다(H)      │
│                                            │
│  "뉴스는 당신이 사는 순간, 이미 늦습니다."   │
│                                            │
│  장 마감 후 회고 리포트를 보내드립니다.      │
│                                            │
│            [ 확인 ]                         │
└────────────────────────────────────────────┘
```

**톤앤매너**:
- Solid 톤: "오늘 당신은 워렌 버핏이었나요, 아니면 불나방이었나요?"
- DÉJÀ BUY 톤: **"오늘 당신은: 🔥 NFH - 불나방"**
  - 더 짧고, 더 직접적.
  - 질문이 아니라 **선언**.
  - "당신은 이랬다." 팩트로 말한다.

오늘 매수 패턴 분석 완료.

## 7. 무료 vs 프리미엄 분리

### 무료 기능

1. **3초 태깅** - 매수 시 페르소나 기록
2. **장 마감 후 당일 회고** - 오늘 매수 패턴 + 결과
3. **최근 10건 기록 조회** - 기록 탭에서 확인
4. **기본 통계** - 유형별 횟수, 최근 1개월 분포
5. **대표 유형 1개** - "당신은 불나방입니다"

### 프리미엄 기능 (DÉJÀ BUY Pro)

1. **무제한 기록 조회** - 전체 매수 이력
2. **고급 통계** - 유형별 승률, 평균 수익률, 3개월/6개월/1년 추이
3. **월간 회고 리포트** - 매월 1일, 지난달 패턴 종합 분석
4. **알림 커스터마이징** - 특정 유형 매수 시 경고 알림 설정
   - 예: "N(뉴스) 유형 선택 시 → '당신은 뉴스형일 때 승률 10%입니다' 경고"
5. **페르소나 변화 그래프** - 시계열 그래프로 성장 추이 시각화
6. **CSV 다운로드** - 전체 매수 기록 데이터 다운로드

**가격**: 월 4,900원 또는 연 49,000원 (월 환산 4,083원)

## 9. 기술 요구사항

### 데이터베이스 스키마

**신규 테이블: `persona_records`**

```sql
CREATE TABLE persona_records (
  id BIGSERIAL PRIMARY KEY,
  user_id BIGINT NOT NULL,

  -- 매수 정보
  asset_code VARCHAR(50) NOT NULL,
  asset_name VARCHAR(100) NOT NULL,
  buy_price DECIMAL(15,2) NOT NULL,
  buy_amount DECIMAL(15,2) NOT NULL,
  buy_timestamp TIMESTAMP NOT NULL,

  -- 페르소나 태깅
  why_code VARCHAR(1) NOT NULL, -- N/C/F
  time_code VARCHAR(1) NOT NULL, -- S/W/H
  risk_code VARCHAR(1) NOT NULL, -- H/L
  persona_code VARCHAR(3) NOT NULL, -- 조합 (NFH 등)
  persona_name VARCHAR(50) NOT NULL, -- 별명 (불나방 등)

  -- 결과 (장 마감 후 업데이트)
  close_price DECIMAL(15,2),
  result_rate DECIMAL(5,2), -- 수익률 %
  result_status VARCHAR(10), -- win/lose/draw

  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

CREATE INDEX idx_persona_user ON persona_records(user_id);
CREATE INDEX idx_persona_code ON persona_records(persona_code);
CREATE INDEX idx_persona_date ON persona_records(buy_timestamp);
```

## 13. 리스크 및 대응

| 리스크 | 영향도 | 대응 방안 |
|--------|--------|----------|
| 3초 태깅이 귀찮아서 건너뜀 | 높음 | 스킵 버튼 없음. 태깅 안 하면 기록 자체가 안 됨. 강제성 부여. |
| 회고 리포트를 안 봄 | 중간 | 푸시 문구에 숫자 포함. "승률 10%" 같은 임팩트 있는 정보 선공개 |
| 유형이 너무 많아서 헷갈림 | 낮음 | 사용자에게는 대표 유형 1개만 강조. 18가지는 백엔드에서만 관리 |
| 프리미엄이 비싸다고 느낌 | 중간 | 무료로도 핵심 가치 제공. 프리미엄은 "더 자세히" 보는 용도 |

## 15. 참고: 원본(Solid) vs DÉJÀ BUY 톤 비교

| 구분 | Solid 톤 | DÉJÀ BUY 톤 |
|------|----------|-------------|
| 질문 방식 | "오늘 당신은 워렌 버핏이었나요, 아니면 불나방이었나요?" | "오늘 당신은: 🔥 NFH - 불나방" |
| 피드백 톤 | "회원님은 뉴스를 볼 때마다 고점에서 삽니다." | "당신은 뉴스를 볼 때마다 고점에서 삽니다." |
| 조언 방식 | "다음에 뉴스가 보이면, Solid의 '참음 기록'을 먼저 눌러보세요." | "다음에 뉴스가 보이면, '참는다' 버튼을." |
| 별명 | 불나방, 분석가, 도박사 (동일) | (동일하지만 더 짧고 직접적) |
| 전체 톤 | 교육적, 친절 | 냉정한 팩트, 직접적 |

**핵심 차이**:
- Solid: "회원님"이라는 존칭 사용, 질문형 문장, 설명이 긴 편
- DÉJÀ BUY: "당신"이라는 2인칭, 선언형 문장, 짧고 강한 마침

## 16. 마무리: 페르소나가 DÉJÀ BUY에 주는 가치

### 기존 DÉJÀ BUY의 철학

> "또 사려고? 이거 전에도 봤잖아."

차트 패턴은 반복된다. FOMO는 반복된다.
DÉJÀ BUY는 과거의 차트를 보여주며 말한다. **"이거 전에 본 패턴인데?"**

### 페르소나가 추가하는 가치

> "또 이 유형으로 샀구나. 이거 전에도 봤잖아."

차트만 반복되는 게 아니다. **당신의 매수 패턴도 반복된다.**
페르소나는 당신의 패턴을 보여주며 말한다. **"너는 또 이러네."**

### 방패와 거울

- **방패 (Shadow Portfolio)**: "참았으니까 지켰다"
- **거울 (Persona)**: "왜 못 참았는지 보여줄게"

방패만 있으면 참다가 폭발한다.
거울이 있어야 근본적으로 변한다.

### 최종 슬로건

> **"과거 차트도 Déjà Vu. 당신의 매수 패턴도 Déjà Buy."**
