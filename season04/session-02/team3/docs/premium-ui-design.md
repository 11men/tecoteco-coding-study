## 1. 설계 원칙

### 1.1 데자바이 프리미엄의 철학

```
프리미엄은 "더 많이"가 아니라 "더 깊이"

무료: FOMO를 인식하고 기록한다
프리미엄: FOMO 패턴을 분석하고 예측한다
```

### 1.2 UX 설계 원칙

**원칙 1: 투명하게, 하지만 거슬리지 않게**
- 잠긴 기능은 명확히 보여주되, 무료 경험을 방해하지 않는다
- "이게 있다는 걸 알려주지만, 강요하지 않는다"

**원칙 2: 다크모드 + 글래스모피즘 조화**
- 프리미엄 요소는 블루 악센트(#3B82F6)로 강조
- 잠긴 컨텐츠는 블러 처리 + 반투명 오버레이
- 자물쇠 아이콘은 최소화

**원칙 3: 팩트 기반 설득**
- "프리미엄으로 더 나은 분석"이 아니라
- "프리미엄은 과거 패턴 매칭 89% → 상세 매칭 시나리오 3개 제공"

**원칙 4: FOMO를 역이용하지 않는다**
- "지금 결제하면 50% 할인!" 같은 급한 CTA 금지
- "이 기능이 필요할 때, 여기 있습니다" 톤 유지

## 2. 페이월 UI 패턴

### 2.1 기본 패턴: 3단계 정보 노출

```
[1단계] 티저 (Tease)
  - 기능이 존재한다는 걸 보여줌
  - 무엇을 할 수 있는지 1줄 설명
  - 예: "과거 패턴 상세 매칭" + 아이콘

[2단계] 미리보기 (Preview)
  - 일부 내용을 실제로 보여줌 (흐릿하게)
  - 호기심 유발
  - 예: 시나리오 비교 그래프 블러 처리

[3단계] 잠금 (Lock)
  - "이 기능은 프리미엄입니다" 명확한 표시
  - CTA: "자세히 보기" 또는 "패턴 전체 보기"
```

### 2.2 잠긴 컨텐츠 표현 방법

#### A. 글래스모피즘 오버레이 (기본 패턴)

```
+------------------------------------------+
|  📊 과거 패턴 상세 매칭                   |
|                                          |
|  ╔══════════════════════════════╗       |
|  ║  [블러 처리된 차트]             ║       |
|  ║  • 2024.03 비트코인 패턴        ║       |
|  ║  • 2023.11 이더리움 패턴        ║       |
|  ║  • 2024.08 애플 패턴            ║       |
|  ╚══════════════════════════════╝       |
|       ⬆ 반투명 글래스 오버레이            |
|                                          |
|       ⚡ 프리미엄                         |
|       3개 매칭 시나리오 + 타이밍 분석     |
|                                          |
|       [패턴 전체 보기]                   |
|                                          |
+------------------------------------------+
```

**디자인 스펙:**
- 오버레이 배경: `rgba(0, 0, 0, 0.6)` (다크모드)
- 블러: `backdrop-filter: blur(10px)`
- 테두리: `1px solid rgba(59, 130, 246, 0.3)` (블루 악센트)
- 그라디언트 힌트: 상단 10% 영역은 블러 없이 보임

#### B. 부분 공개 + 페이드아웃 (그라디언트 마스크)

```
+------------------------------------------+
|  📈 시뮬레이션: 3일 기다렸다면?            |
|                                          |
|  현재 매수:  1,000,000원                 |
|  3일 후:    1,150,000원  (+15%) ✅      |
|  7일 후:    ▓▓▓▓▓▓▓▓                    |
|  14일 후:   ▓▓▓▓▓▓▓▓                    |
|  30일 후:   ▓▓▓▓▓▓▓▓                    |
|             ⬆ 페이드 그라디언트           |
|                                          |
|  ⚡ 프리미엄으로 전체 타임라인 보기         |
|                                          |
|  [시뮬레이션 전체 보기]                   |
|                                          |
+------------------------------------------+
```

**디자인 스펙:**
- 첫 2줄: 완전 공개
- 3줄부터: 세로 그라디언트 `linear-gradient(to bottom, rgba(0,0,0,0) 0%, rgba(0,0,0,1) 100%)`
- 텍스트는 남기되, 숫자만 `▓` 문자로 마스킹

#### C. 카운트 제한 UI (홈 화면)

```
+------------------------------------------+
|  🔍 종목 검색                             |
|                                          |
|  테슬라                                   |
```

### 2.3 프리미엄 배지/태그 디자인

#### 타입 A: 인라인 뱃지 (기능명 옆)

```
📊 과거 패턴 상세  ⚡
```

- 아이콘: ⚡ (번개, 프리미엄의 빠른 인사이트 상징)
- 색상: #3B82F6 (블루 악센트)
- 크기: 14px
- 위치: 기능명 우측 4px 간격

#### 타입 B: 코너 리본 (카드 우상단)

```
+------------------------------------------+
|                            ⚡ PREMIUM    |
|  시뮬레이션 비교                          |
|                                          |
|  "3일 기다렸다면 +15%"                    |
|                                          |
+------------------------------------------+
```

- 배경: `linear-gradient(135deg, #3B82F6, #1E3A8A)`
- 텍스트: 흰색, 12px, bold
- 위치: 우상단 코너
- 모양: 라운드 직사각형 `border-radius: 0 8px 0 8px`

#### 타입 C: 풀 배너 (섹션 구분)

```
+------------------------------------------+
|  ⚡ 프리미엄 전용 기능                     |
|  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━  |
|                                          |
|  [블러 처리된 컨텐츠]                      |
|                                          |
+------------------------------------------+
```

- 배경: `rgba(59, 130, 246, 0.1)`
- 텍스트: #3B82F6
- 하단 언더라인: `border-bottom: 1px solid #3B82F6`

## 3. 무료/프리미엄 기능 구분

### 3.1 최종 기능 매트릭스

| 기능 영역 | 무료 | 프리미엄 |
|----------|------|---------|
| **홈 - 패턴 분석** | 월 3회 | 무제한 |
| **패턴 유사도** | 기본 % 표시 | 상세 매칭 3개 + 신뢰도 |
| **시뮬레이션** | 없음 | "3일/7일/14일 기다렸다면" |
| **과거 매칭 상세** | 없음 | 과거 차트 오버레이 + 당시 뉴스 |
| **기록 - Shadow Portfolio** | 전체 | 전체 |
| **JOMO 계산기** | 기본 (치킨, 커피 등) | 맞춤 비유 (내 목표 물건 기준) |
| **타이밍 알림** | 없음 | AI 타이밍 알림 |
| **리포트** | 주간 요약 | 월간 상세 + 연간 리포트 |
| **페르소나** | 기본 유형 (4가지) | 전체 16가지 + 맞춤 조언 |
| **레벨링** | 전체 | 전체 + 프리미엄 전용 뱃지 |
| **광고** | 있음 | 없음 |

### 3.2 프리미엄 가격

**월간 구독: 9,900원/월**
**연간 구독: 89,000원/년** (월 7,417원, 25% 할인)

**가설:** 번개가 "빠른 인사이트" 느낌으로 가장 적합

**B안:** "패턴 전체 보기"
