## 1. 프리미엄 플랜 이름

DÉJÀ BUY의 프리미엄 플랜은 "참음의 달인"을 위한 공간입니다. 이름 자체가 정체성이 되어야 합니다.

### 최종 추천안 (우선순위순)

#### 1순위: **SOLID**
- **의미**: 흔들리지 않는 투자자. 현두님 PRD의 핵심 컨셉.
- **브랜드 연결**: DÉJÀ BUY(문제 인식) → SOLID(해결책/정체성)
- **카피 예시**: "SOLID 멤버는 흔들리지 않습니다."
- **톤**: 진지하지만 무겁지 않음. 자부심.

#### 2순위: **돌부처 플랜 / Stone**
- **의미**: 레벨 체계의 최고 칭호 중 하나. 시장이 흔들려도 움직이지 않는 자.
- **브랜드 연결**: 기존 레벨링 시스템과 자연스럽게 연결
- **카피 예시**: "돌부처만 볼 수 있는 진실."
- **톤**: 유머 + 자조 + 자부심

#### 3순위: **냉장고 플랜 / Fridge**
- **의미**: 투자 격언 "주식은 냉장고에 넣어두고 잊어라". 장기 투자의 인내심.
- **브랜드 연결**: 참을성 = 차가운 이성
- **카피 예시**: "냉장고에 넣고 잊은 사람만 삽니다."
- **톤**: 일상 언어 + 위트

#### 4순위: **다이아몬드 핸즈 / 💎**
- **의미**: 투자 커뮤니티 용어. 하락장에도 팔지 않는 강철 멘탈.
- **브랜드 연결**: 이미 뱃지 시스템에 있음. 친숙함.
- **카피 예시**: "다이아몬드 핸즈의 특권."
- **톤**: 커뮤니티 친화적

#### 5순위: **DÉJÀ PRO**
- **의미**: 프로페셔널한 과거 분석가. 반복을 읽는 자.
- **브랜드 연결**: 브랜드명 직접 활용
- **카피 예시**: "전에 본 패턴, 이번엔 이긴다."
- **톤**: 직관적이지만 평범할 수 있음

## 2. 무료 플랜 이름

무료는 "시작"이 아니라 "아직 흔들리는 단계"로 프레이밍. 성장 욕구 자극.

### 최종 추천안

#### 1순위: **펄럭귀**
- **의미**: 레벨 1 칭호. 귀가 얇아 뉴스에 흔들리는 초보 투자자.
- **브랜드 연결**: 기존 레벨링 시스템 + 자조적 유머
- **카피 예시**: "펄럭귀 단계. 지금은 여기까지."
- **톤**: 비난 아닌 인정. "나도 그랬어" 공감대

#### 2순위: **갈대**
- **의미**: "흔들리는 갈대" 레벨 1 칭호. 시장에 쉽게 흔들림.
- **브랜드 연결**: 레벨 체계 직접 활용
- **카피 예시**: "갈대는 바람에 흔들립니다. 아직은."
- **톤**: 시적이지만 직관적

#### 3순위: **관망**
- **의미**: 지켜보기만 하는 단계. 본격적 참여 전.
- **브랜드 연결**: 중립적 표현
- **카피 예시**: "관망 중. 더 보려면 SOLID."
- **톤**: 부담 없음

### DÉJÀ BUY 카피 작성 원칙

#### ✅ DO (해야 할 것)

1. **친구처럼 직접적**
   - "또 사려고?" "어차피 또 분석할 거잖아."
   - 반말 톤 허용 (과하지 않게)

2. **잔소리 아닌 팩트**
   - "참으세요" (X) → "뉴스형 승률 12%" (O)
   - 숫자와 데이터가 말하게

3. **자조적 위트**
   - "불나방", "팔랑귀" 같은 유머러스한 별명
   - 자기비판이 아닌 자기관찰의 톤

4. **짧고 강렬하게**
   - 장황한 설명 지양
   - 핵심 메시지 1-2문장으로

#### ❌ DON'T (하지 말아야 할 것)

1. **과도한 위로**
   - "괜찮아요~", "화이팅!" (X)
   - DÉJÀ BUY는 위로가 아니라 직면

2. **너무 가벼움**
   - "ㅋㅋㅋ", "ㅎㅎ" 같은 인터넷 용어 지양
   - 유머는 있지만 가볍지는 않게

3. **종목 추천/시그널**
   - 절대 금지
   - 법적 리스크 + 브랜드 정체성 훼손

4. **강요하는 톤**
   - "지금 구독하세요!" (X)
   - "어차피 또 분석할 거잖아." (O)
