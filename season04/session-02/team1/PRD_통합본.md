## 1. 문제 정의

### 1.1 현재 문제 상황

투자자들이 겪는 공통적인 악순환:

**A. FOMO(Fear of Missing Out)의 악순환**

```
"지금 안 사면 기회를 놓친다"는 불안감
        |
SNS/커뮤니티에서 "XX% 올랐다" 목격
        |
조급함 → 고점에 매수
        |
하락 → 손실 → 후회
        |
다음에도 반복...
```

**B. 충동 매매의 반복**

- 감정적으로 매수했다가 후회
- "참았어야 했는데..."라는 생각은 하지만 다음에도 반복
- 매수 버튼을 누르는 순간은 기억하지만, **참은 성공은 인식하지 못함**

**C. 자기 통제력 부족에 대한 자책**

- "나는 왜 참지 못할까?"
- "또 충동적으로 샀어..." -> 자존감 하락
- 하지만 참을 수 있는 **구체적인 방법이 없음**

### 1.2 기존 솔루션의 한계

| 기존 방법 | 문제점 |
|-----------|--------|
| 투자 원칙 세우기 | 막상 FOMO 올 때는 원칙이 무너짐 |
| 가상 매매 (모의투자) | 수익 내는 게 목표라 FOMO 극복 훈련 안 됨 |
| 투자 일기 | 귀찮아서 안 씀, 피드백 없음 |
| 알림 꺼두기 | 근본 해결 아님, 정보 단절 |
| 자기 다짐 | 감정 앞에서 무력 |

**핵심 문제**: 기존 솔루션은 "안 하는 것"에 대한 보상이 없다. 뇌는 "안 한 행동"을 인식하지 못한다.

### 1.3 우리의 해결 방향

```
매수하고 싶다
      |
  (기존) 그냥 참기 --> 뇌: 아무것도 안 함 --> 보상 없음
      |
  (우리) 섀도우 포트폴리오 기록 --> 뇌: 뭔가 했다! --> 즉각 만족감
      |
  1주일 후 결과 확인
      |
  [하락 시] "치킨 50마리 방어!" --> 성공 경험
  [상승 시] "원칙을 지켰습니다" --> 교육 피드백
```

**"참음"을 능동적 행동으로 전환하고, 게임적 보상을 부여한다.**

## 2. 제품 개요 및 핵심 가치 제안

### 2.1 제품명 (최종 후보)

- **Shadow Portfolio** (섀도우 포트폴리오)
- **SeedGuard** (시드가드)
- **마이너스의 손** (Minus Touch)

### 2.2 제품 정의

사용자가 FOMO를 느끼는 순간 실제 매수 대신 **"섀도우(그림자) 매수"**를 기록하고, 1주일 후 실제 주가 변동을 추적하여:

1. **하락 시**: 방어한 손실을 "치킨 50마리" 같은 실물 가치로 환산하여 축하
2. **상승 시**: 원칙을 지킨 것을 칭찬하고 교육적 피드백 제공
3. **참음에 대한 보상**: 시드머니(가상 재화) 지급
4. **가상 투자 연습**: 모은 시드로 리스크 없는 투자 경험

### 2.3 한 문장 요약

> "매수 버튼을 누르는 대신 기록 버튼을 눌러서, 참음을 게임처럼 즐기고, 손실 방어를 축하받으며, FOMO를 극복하는 습관을 만드는 서비스"

### 2.4 사용자가 얻는 5가지 가치

#### (1) 즉각적 보상: "참은 것"을 행동으로 전환

- 매수 대신 섀도우 기록 = "뭔가 했다!"는 뇌의 보상
- 기록 즉시 시드머니 획득
- 참음이 곧 게임 내 자산 축적

#### (2) 심리적 보상: 손실 방어를 구체적 이득으로 시각화

- 추상적인 "손해 안 봄" -> "치킨 50마리를 지켰습니다"
- JOMO Calculator로 실물 가치 환산
- "안 산 게 아니라 벌은 것" 프레이밍

#### (3) 게임적 재미: 참을성을 수치화하고 레벨업

- 레벨, 배지, 스트릭, 시드머니
- "다음 레벨까지 3일 남았다" -> 매수 충동을 참을 동기
- 지루한 인내 -> 재미있는 도전

#### (4) 장기적 습관 형성: FOMO 극복 근육 키우기

- 1주일 사이클 반복으로 인내심 훈련
- 감정 체크인으로 자기 통제력 향상
- 투자 결정을 감정이 아닌 데이터로 바라보는 습관

#### (5) 투자 연습: 리스크 없는 투자 경험

- 모은 시드로 가상 투자 실전 연습
- 소수점 매매로 다양한 종목 경험
- 실패해도 실제 돈 손실 없음

### 2.5 차별화 포인트

| 기존 모의투자 | Shadow Portfolio |
|--------------|------------------|
| 가상 수익 중심 | 손실 방어 중심 |
| 차트 분석 기반 | 심리 개선 기반 |
| 수익 경쟁 | 절제 경쟁 |
| 숫자 중심 | 실물 가치 시각화 |
| 게임적 요소 없음 | 레벨/배지/스트릭 |

## 3. 타겟 유저

### 3.1 Primary 타겟

**"충동 매수 경험이 있는 2030 투자 초중급자"**

| 속성 | 내용 |
|------|------|
| 연령대 | 20~35세 |
| 투자 경험 | 6개월 ~ 3년 |
| 투자 대상 | 주식, 코인, ETF |
| 공통 경험 | FOMO로 고점 매수 후 후회한 경험 |
| 디지털 친화도 | 높음 (앱/게임 경험 풍부) |
| 투자 지식 | 기본~중급 (차트 읽기, 기본 분석 가능) |
| 커뮤니티 활동 | 주식/코인 커뮤니티, 오픈채팅 참여 |

### 3.2 통합 페르소나

**페르소나 A: 김민수 (27세, 직장인) — "충동 매수 반성러"**

- 코인 투자 1년 차
- 월급 300만 원 중 50만 원 투자
- SNS에서 "XX코인 100% 올랐다" 보면 바로 매수
- 3번 연속 고점 매수 -> 총 80만 원 손실
- "나는 왜 매번 이러지..." 자책 중
- **니즈**: "내가 참았던 게 맞는 선택이었는지 확인하고 싶다"
- **게임 동기**: "나 오늘 모의투자로 150만원 잃었어 ㅋㅋ" → SNS 공유 본능

**페르소나 B: 이수진 (31세, 프리랜서) — "자기 통제 챌린저"**

- 주식 투자 2년 차
- 유튜브 투자 채널 5개 구독
- 급등주 뉴스 보면 "이번엔 다르다" 매수
- 이성적으로는 알지만 감정을 통제 못 함
- 투자 일기 시도했으나 3일 만에 포기
- **니즈**: "참는 기록을 데이터로 보고 싶다", "목표(아이폰, 여행)를 위해 저축하고 싶다"
- **게임 동기**: 랭킹 경쟁보다 자기 기록 추적

### 3.3 Secondary 타겟

- **투자 입문자**: 실제 투자 전 가상 투자로 연습하고 싶은 사람
- **투자 교육자**: 학생/수강생에게 투자 심리 훈련 도구로 활용

### 4.1 섀도우 포트폴리오 기록

#### 기능 설명

FOMO를 느끼는 순간, 실제 매수 대신 "섀도우(그림자) 매수"를 기록한다.

#### 입력 항목

| 항목 | 필수 여부 | 설명 |
|------|-----------|------|
| 종목명 | 필수 | 검색으로 선택 (국내/해외 주식, 코인, ETF) |
| 매수 희망 금액 | 필수 | 원화 단위 입력 |
| FOMO 강도 | 필수 | 1~5 단계 (1: 살까 말까 / 5: 지금 안 사면 죽을 것 같다) |
| 메모 | 선택 | "왜 사고 싶은지" 자유 기록 |
| 참은 이유 | 선택 | 선택지 제공 (고점 우려 / 원칙 지키기 / 자금 부족 / 기타) |

#### 기록 시점

- 실시간 가격(종가) 자동 스냅샷 저장
- 1주일 후 가격 자동 추적
- 가격 변동률 자동 계산

#### 가상 잔고 시스템

```
신규 가입 시   → 가상 잔고 1,000,000원 지급 (1회)
섀도우 기록 시 → 가상 잔고에서 차감
광고 시청     → 가상 잔고 1,000,000원 재충전
```

- 가상 잔고는 서버 사이드 관리 (조작 방지)
- **마이너스 잔고 허용** (음수 표시 가능)
- 실제 돈과 교환 불가 (사행성 규정 회피)

#### UI 플로우

```
[메인 화면]
     |
[+ 섀도우 기록하기] 버튼
     |
[종목 검색] -> 종목 선택
     |
[금액 입력] -> "100만 원"
     |
[FOMO 강도 선택] -> 4/5
     |
[메모 입력 (선택)]
     |
[기록 완료!]
  "참음 +1, 시드 +5,000원 획득"
  "24시간 참음 체크인이 시작됩니다!"
```

### 4.3 목표 설정 기능 (신규 통합)

사용자가 방어한 돈으로 살 수 있는 목표 물건을 설정할 수 있다.

#### 기능 설명

- 목표 물건 선택 (아이폰, 해외여행, 노트북, 명품 가방 등)
- 목표 금액 자동 설정
- 누적 방어 금액 대비 달성률 시각화
- 목표 달성 시 특별 배지 지급

#### UI 예시

```
+------------------------------------------+
|                                          |
|  내 목표                                  |
|                                          |
|  🎯 아이폰 16 Pro                         |

**설계 원칙:**
- 방치 방지: 단순 시간 경과가 아닌, 사용자의 액션이 필요
- 자기 성찰: 체크인 자체가 감정 인지 훈련
- 부담 최소화: 3초면 완료

**체크인 주기**: 4시간마다 (하루 최대 6회)

**체크인 기간**: 섀도우 기록 후 24시간

**체크인 방식**: 감정 체크 (3초 완료)

```
+------------------------------------------+
|                                          |
|  참음 체크인                              |
|                                          |
|  비트코인 섀도우                           |
|  참은 지 4시간 경과                        |
|                                          |
|  지금 기분은 어떤가요?                     |
|                                          |
|  (1) 여유롭다                             |
|  (2) 조금 불안하다                         |
|  (3) 아직도 사고 싶다                      |
|                                          |
|  -> 선택 시 5,000원 획득                   |
|  -> 감정 데이터 축적                       |
|                                          |
+------------------------------------------+
```

**패널티**: 해당 회차 리워드만 손실 (가벼운 패널티)
- 체크인 놓침 -> 해당 5,000원만 못 받음
- 다음 체크인 기회는 유지
- 연속 스트릭은 유지 (끊기지 않음)

**24시간 참음 리워드 합계: 최대 35,000원**
(기록 시 5,000원 + 체크인 6회 x 5,000원)

#### 4.6.4 시드머니 활용: 가상 투자 포트폴리오

획득한 시드로 실시간 가격 연동 가상 투자 연습.

**핵심 특징:**
- **소수점 매매 지원**: 0.001 BTC, 0.1주 등 소액 투자 가능
- **실시간 가격 연동**: 실제 시장 가격과 동일
- **수익/손실 반영**: 가상 포트폴리오에 실시간 반영
- **매수/매도 자유**: 가상이므로 수수료 없음

```
목표: 본격 수익화

  4. 금융사 제휴 수수료 (CPA)

### 7.2 주요 수익원 상세

#### (1) 광고 (핵심 수익원 - Phase 1)

| 트리거 | 광고 유형 | 보상 |
|--------|----------|------|
| 가상 잔고 소진 | 보상형 전면 광고 (30초) | 가상 100만원 재충전 |
| 시드 획득 (선택) | 보상형 광고 | 시드 +5,000원 |
| 챌린지 참여 | 선택형 광고 시청 | 추가 기록 슬롯 |
| 결과 카드 공유 전 | 건너뛸 수 없는 5초 광고 | 공유 실행 |

**광고 수익 추정 (보수적)**

```
시나리오 1: DAU 1,000명
  1,000명 × 일 평균 2회 광고 시청 × eCPM 3,000원 / 1,000회
  = 일 수익 약 6,000원 → 월 약 180,000원

시나리오 2: DAU 15,000명
  15,000명 × 일 평균 2회 광고 시청 × eCPM 3,000원 / 1,000회
  = 일 수익 약 90,000원 → 월 약 2,700,000원

MAU 50,000명 (DAU 15,000) 달성 시:
  월 약 2,700,000원

MAU 100,000명 달성 시:
  월 약 5,400,000원 예상
```

#### (2) 현금 충전 (시드 구매) - 핵심 수익원 (Phase 2)

```
비율: 1:5 (현금 10,000원 -> 시드 50,000원)
결제 단위: 1,000원 / 5,000원 / 10,000원 / 50,000원 / 100,000원
결제 수단: 신용카드, 카카오페이, 네이버페이, 앱스토어/구글플레이

예상 ARPPU: 월 15,000원
유료 전환율: 5~10%
MAU 10,000명 기준: 월 750만원 ~ 1,500만원
MAU 50,000명 기준: 월 3,750만원 ~ 7,500만원
```

**전환 포인트:**
- "시드가 부족합니다. 충전하시겠습니까?"

- 초대 10명 한도 도달 후: "더 많은 시드가 필요하신가요?"

## 9. 법적 고려사항

### 9.1 시드머니 법적 정의

```
시드머니는 앱 내에서만 사용 가능한 가상 재화입니다.
  - 실제 현금으로 전환 불가
  - 타인에게 양도 불가 (Phase 1)
  - 현금 충전 시 환불 정책 별도 명시
  - 전자상거래법상 "디지털 콘텐츠" 분류
```

### 9.2 가상 투자 면책

```
본 서비스의 투자 기능은 가상 시드머니를 활용한 교육 목적의 모의 투자입니다.
  - 실제 금융 상품 거래가 아님
  - 투자 조언/추천이 아님
  - 실제 투자 시 발생하는 손실에 대해 책임지지 않음
  - 금융투자업 등록 불필요 (가상 거래)

앱 내 상시 표시:
"본 앱은 투자 조언을 제공하지 않습니다.
모든 기록은 사용자 개인의 가상 기록이며,
실제 투자 결과와 다를 수 있습니다."
```

### 9.3 사행성 규정 회피

```
구조적 해소:
  - 가상 잔고는 게임 내 재화 — 현금 전환 불가
  - 상금 없음 — 배지·레벨 등 비금전 보상만 지급
  - 실제 투자 조언 아님 — 면책 조항 명시 필수
```

### 9.4 개인정보 보호

```
수집 정보:
  - 이메일, 닉네임 (회원가입)
  - 투자 관심 종목 (서비스 제공)
  - 감정 체크인 데이터 (서비스 개선)
  - 앱 이용 패턴 (서비스 개선)
  - 건강 데이터 (만보기, 선택 사항)

보호 원칙:
  - 개인정보보호법 준수
  - 데이터 익명화 처리 (분석/판매 시)
  - 사용자 동의 기반 수집
  - 데이터 삭제 요청 권리 보장
```

### 9.5 광고 규제

```
허용 광고:
  - 투자 교육 콘텐츠
  - 재무설계 상담
  - 자산관리 앱
  - 증권사/거래소 (제휴)

금지 광고:
  - 리딩방, 시그널 서비스
  - 고수익 보장 투자 상품
  - 레버리지/파생 상품
  - 불법 금융 상품

이유: 서비스의 본질(충동 매매 방지)과 충돌하는 광고 배제
```

## 13. 부록

### 부록 A: 경쟁사 분석

| 서비스 | 특징 | 우리와의 차이 |
|--------|------|--------------|
| 토스증권 모의투자 | 가상 머니로 실전 투자 연습 | 수익 추구 목적. FOMO 극복 관점 없음 |
| 트레이딩뷰 | 차트 분석, 페이퍼 트레이딩 | 전문가 도구. 심리 훈련 요소 없음 |
| 증권사 모의투자 | 가상 계좌로 투자 체험 | 투자 실력 향상 목적. 심리 개선 없음 |
| 투자 일기 앱 | 투자 결정 기록 | 피드백/보상 없음. 지속 동기 부족 |

**우리의 차별점:**
1. "참음"에 특화된 유일한 서비스
2. 게이미피케이션으로 지속 동기 부여
3. 손실 방어를 실물 가치로 시각화
4. 시드머니 -> 가상 투자 선순환 구조
5. 감정 체크인으로 자기 성찰 유도
6. 목표 설정으로 구체적 동기 부여
7. 역발상 랭킹 옵션 (유머러스한 바이럴)
8. 건강 연동으로 전인적 웰빙 추구
