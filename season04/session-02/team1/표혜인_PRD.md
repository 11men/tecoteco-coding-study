# SeedGuard -- 참아서 번 돈

## 행동경제학 기반 투자 심리 게이미피케이션 서비스 PRD

------------------------------------------------------------------------

## 1. 제품 개요

### 1.1 제품명 (가칭)

**SeedGuard**

### 1.2 제품 정의

사용자가 충동 매수를 실제로 실행하는 대신 가상 매수로 기록하고,\
하락 시 방어한 손실 금액을 Seed(재화)로 환산하여 보상받는\
행동교정 기반 투자 심리 게임형 서비스.

또한 방어 금액을 실물 소비 가치로 환산하여 체감도를 높인다.

------------------------------------------------------------------------

## 2. 핵심 컨셉

> 사지 않아서 지킨 돈은, 이미 번 돈이다.

> 숫자가 아닌 '물건'으로 보여준다.

------------------------------------------------------------------------

## 3. 문제 정의

-   FOMO로 인한 추격 매수 발생
-   손실 회피 체감이 약함
-   숫자는 추상적이라 감정적 인지가 낮음
-   소비 가치로 환산하면 체감도가 상승

------------------------------------------------------------------------

## 4. 주요 기능

### 4.1 가상 매수 기록

-   종목명 검색
-   기록 시점 가격 자동 입력
-   가상 매수 금액 입력
-   감정 상태 선택 (FOMO, 불안, 확신 등)
-   기록 시간 저장

------------------------------------------------------------------------

### 4.2 손실 방어 계산

하락 시:

방어 금액 = 하락률 × 가상 매수 금액

예: - -12% - 200만원 가상 매수 → 24만원 방어 성공

상승 시: - Seed 미지급 - 기회비용 기록만 저장

------------------------------------------------------------------------

### 4.3 실물 환산 시각화 기능

방어 금액을 대표 소비재와 매칭하여 카드 형태로 표시

예시 매칭 테이블:

  금액대         대표 물건
  -------------- -----------------
  5,000원        커피 1잔
  30,000원       치킨 1마리
  100,000원      백화점 상품권
  300,000원      에어팟
  1,000,000원    스마트폰
  2,000,000원    해외여행 항공권
  5,000,000원    노트북
  10,000,000원   명품 가방

예시 문구:

"당신은 24만원을 지켰습니다."\
"에어팟 1개 + 치킨 3마리를 살 수 있는 금액입니다."

------------------------------------------------------------------------

### 4.4 Seed 시스템

1,000원 = 1 Seed

획득 방법: - 손실 방어 성공 - 만보기 연동 - 광고 시청 - 출석 보상

사용처: - 프로필 커스터마이징 - 심리 리포트 열람 - 광고 제거 - 프리미엄
통계

------------------------------------------------------------------------

### 4.5 랭킹 시스템

랭킹 기준: - 월간 방어 금액 - 누적 방어 금액 - 연속 방어 일수 - 목표
물건 달성률

SNS 공유 카드 자동 생성 기능 포함

------------------------------------------------------------------------

### 4.6 목표 설정 기능

사용자가 목표 물건 설정 가능

예: - 아이폰 - 해외여행 - 노트북

목표 대비 달성률 시각화 제공

------------------------------------------------------------------------

## 5. 사용자 흐름

1.  급등 종목 발견
2.  가상 매수 기록
3.  일정 기간 경과
4.  방어 성공 판정
5.  방어 금액 계산
6.  실물 환산 카드 표시
7.  Seed 지급
8.  랭킹 반영
