# 상업화 통보 템플릿

프로젝트를 상업화하려고 할 때 팀원들에게 보내는 메시지 템플릿입니다.

---

## 템플릿 1: 간단한 버전

가장 기본적인 통보 메시지입니다. 간단하게 알리고 싶을 때 사용하세요.

```
제목: Session X 프로젝트 상업화 계획 안내

안녕하세요, 팀원 여러분.

Session X에서 함께 작업한 "[프로젝트명]" 아이디어를
저 혼자 실제 앱으로 개발해보려고 합니다.

함께 고민해준 것 감사드리고,
앱 출시 시 크레딧에 명시하겠습니다.

응원해주세요!

[본인 이름] 드림
```

---

## 템플릿 2: 보상 제안 버전

팀원들에게 감사 표시 옵션을 제공하고 싶을 때 사용하세요.

```
제목: Session X 프로젝트 상업화 및 보상 제안

안녕하세요, 팀원 여러분.

Session X에서 함께 기획한 "[프로젝트명]" 아이디어로 
실제 서비스를 만들어보려고 합니다.

초기 기획에 함께해주신 감사의 표시로
다음 중 하나를 선택해주시면 감사하겠습니다:

1. 각자 [금액]만원 감사금
2. 각자 [%]% 지분 (법인 설립 시)
3. 크레딧 표기만 (금전 없음)
4. 아무것도 원하지 않음

30일 내 답변 부탁드립니다.
답변 없으면 3번으로 진행하겠습니다.

궁금하신 점이나 의견 있으시면 편하게 연락 주세요.

[본인 이름] 드림
```

---

## 템플릿 3: 함께 하자는 제안 버전

팀원들에게 공동 창업을 제안하고 싶을 때 사용하세요.

```
제목: Session X 프로젝트 - 함께 만들어가실 의향 있으신가요?

안녕하세요, 팀원 여러분.

Session X에서 함께 만든 "[프로젝트명]" 아이디어가
실제로 가치 있을 것 같아 제품화를 고민하고 있습니다.

혹시 함께 만들어가실 의향이 있으신가요?

**함께 하신다면:**
- 역할: [예상 역할 분담]
- 지분: [기여도 기반 배분 예정]
- 시간: [예상 개발 기간]

**참여가 어려우시다면:**
소액 감사 표시로 크레딧 표기나 소액 지분을 드릴 수 있습니다.

30일 내 의견 주시면 감사하겠습니다.

[본인 이름] 드림
```

---

## 작성 시 주의사항

### 반드시 포함할 내용
- Session 번호와 프로젝트명
- 상업화 계획 (혼자인지, 팀인지)
- 크레딧 표기 약속
- 연락처 또는 답변 방법

### 선택적으로 포함할 내용
- 보상 제안 (감사금, 지분 등)
- 답변 기한 (보통 30일 권장)
- 향후 계획 (출시 시기, 플랫폼 등)

### 피해야 할 표현
- "허락해주세요" (통보이지 허가가 아님)
- "권리를 포기해주세요" (법적으로 민감)
- "절대 지분 안 줌" (너무 단정적)

## 발송 방법

### 이메일로 보내기
- 제목을 명확하게
- 참조에 테코테코 리더 포함 (선택)
- 중요 표시

### 메신저로 보내기
- Discord #commercial 채널 활용
- 또는 개인 DM
- 단체 채팅방에서 전달

### 문서로 남기기
- Google Docs나 Notion에 작성
- 링크를 공유
- 날짜와 발신자 명확히

## 예시 케이스

### 케이스 1: 차승민님이 혼자 개발하는 경우

```
안녕하세요, 김지연님, 박현두님.

Session 01에서 함께 만든 "파업 출퇴근 비서" 아이디어를
제가 혼자 앱으로 개발해보려고 합니다.

React Native로 3개월 정도 개발할 계획이고,
앱 출시 시 About 페이지에 테코테코와 팀원분들을
크레딧으로 명시하겠습니다.

응원해주시면 감사하겠습니다!

차승민 드림
```

```

Session 01 "파업 출퇴근 비서"를 제가 계속 개발하려 합니다.

감사의 표시로 다음 중 선택해주세요:
1. 각자 50만원 감사금
2. 각자 5% 지분
3. 크레딧 표기만

30일 내 답변 주시면 감사하겠습니다.

홍영훈 드림
```
