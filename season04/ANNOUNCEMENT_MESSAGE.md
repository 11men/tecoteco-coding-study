```

안녕하세요, 테코테코 여러분!

이제 명확한 가이드라인이 있습니다! 🎉

✅ 핵심 내용:
- 누구든 스프린트 아이디어로 상업화 가능
- 실제 개발한 사람이 주된 소유자
- 최소 의무: 팀원 통보 + 크레딧 표기
- 감사 표시는 선택사항 (권장)

📖 자세한 내용:
https://github.com/[repo]/season04/PROJECT_OWNERSHIP_POLICY.md

## 버전 2: 자세한 공지

```
📢 [중요] 테코테코 프로젝트 상업화 정책 안내

안녕하세요, 테코테코 시즌4 참가자 여러분!

여러분이 스프린트에서 만든 아이디어를 실제 제품으로 발전시키고 싶으신가요?
테코테코는 상업화를 적극 독려합니다! 🚀

하지만 함께 만든 아이디어이기에, 최소한의 예의와 배려가 필요하다고 생각합니다.
그래서 명확한 정책을 만들었습니다.

━━━━━━━━━━━━━━━━━━━━━━

🎯 핵심 원칙

1️⃣ 누구든 상업화 가능
   - 혼자도, 팀으로도, 새로운 멤버와도 OK
   - 테코테코는 IP 주장 안 함

2️⃣ 실행자 우선
   - 실제로 개발한 사람이 주된 소유자
   - 법적으로도 새로 작성한 코드는 작성자 소유

3️⃣ 최소 의무 (필수)
   ✅ 팀원 통보: "제가 이거 만들게요" 알리기
   ✅ 크레딧 표기: 앱에 "Tecoteco" 명시

4️⃣ 권장 사항 (선택)
   💝 소액 감사금, 지분, 또는 나중에 보답
   → 강제는 아니지만 아름다운 일입니다

━━━━━━━━━━━━━━━━━━━━━━

📖 전문 읽기:
https://github.com/[repo]/season04/PROJECT_OWNERSHIP_POLICY.md

📝 통보 템플릿:
https://github.com/[repo]/season04/template/commercialization_notice.md

━━━━━━━━━━━━━━━━━━━━━━

💬 자주 묻는 질문

Q: 다른 팀 아이디어도 써도 되나요?
A: 네! 크레딧만 표기하면 됩니다.

Q: 팀원이 반대하면요?
A: 법적으로는 의무 없지만 대화를 권장합니다.

Q: 수익이 나면 테코테코에 기부해야 하나요?
A: 의무는 없어요. 하지만 감사의 표시는 환영합니다 😊

━━━━━━━━━━━━━━━━━━━━━━

하지만 참고용으로 읽어보시면 좋을 것 같습니다!

궁금한 점이나 의견이 있으면 댓글로 알려주세요.
또는 저에게 직접 연락 주셔도 됩니다.

함께 성장하는 테코테코! 🔥
