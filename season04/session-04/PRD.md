# 🔇 소음 지도 (Noise Map) - PRD

## 1. 프로젝트 개요

### 1.1 프로젝트 소개
**소음 지도**는 이사를 준비하는 임차인/세입자가 계약 전에 해당 물건의 소음 정보를 미리 확인할 수 있는 부동산 정보 플랫폼입니다.

### 1.2 핵심 가치 제안
- 🏠 **이사 전 정보 제공**: 계약 후 후회하지 않도록 사전에 소음 환경 파악
- 📊 **객관적 지표**: 주관적 리뷰 + 객관적 데이터(차도, 공사) 결합
- 🗺️ **시각화**: 지도 기반 히트맵으로 직관적인 지역 비교
- 👥 **커뮤니티**: 이웃 정보와 경험 공유

### 1.3 타겟 사용자
- **1차 타겟**: 원룸/오피스텔/아파트 계약을 앞둔 모든 임차인/세입자
- **2차 타겟**: 거주 중이지만 소음으로 인해 이사를 고려하는 사람
- **사용자 특성**:
  - 소음에 민감하거나 조용한 환경을 중요시함
  - 부동산 정보를 꼼꼼히 확인하는 성향
  - 후기와 커뮤니티 정보를 신뢰함

## 2. 핵심 기능

### 2.1 주소 검색 & 소음 리뷰 조회
**목적**: 특정 건물/아파트의 소음 정보를 빠르게 확인

**기능 상세**:
- 주소 검색창 (카카오/네이버 주소 API 자동완성)
- 건물별 소음 리뷰 목록
- 소음 별점 (1~5점) **이미지/차트로 시각화**
- 층별 필터링 (지하, 1층, 2~5층, 6~10층, 11층 이상 등)
- 소음 유형별 태그 필터링
  - 층간소음 (발소리, 의자 끄는 소리)
  - 외부소음 (차량, 공사, 상가)
  - 기타 (엘리베이터, 쓰레기 수거, **반려동물, 아기 울음소리**)

**UI 예시**:
```
[검색창: "서울시 강남구 역삼동..."]
━━━━━━━━━━━━━━━━━━━━━━━━━━━
📍 역삼 아파트 101동
⭐ 소음 평균: 3.2/5.0
📊 [별점 차트 이미지]

🏷️ 필터: [전체] [층간소음] [외부소음] [반려동물] [아기]
📐 층: [전체] [1~5층] [6~10층] [11층+]

━━━━━━━━━━━━━━━━━━━━━━━━━━━
🔊 리뷰 (32개)
└─ 8층 ⭐3.0 #층간소음 #반려동물
   "위층 강아지 소리가 가끔 들립니다..."
```

### 2.2 지도 시각화 (히트맵)
**목적**: 지역별 소음 수준을 한눈에 비교

**기능 상세**:
- 카카오맵/네이버맵 기반 지도
- 소음 평균 점수에 따른 히트맵 (빨강=시끄러움, 초록=조용함)
- 줌 레벨에 따라 시/구/동 단위로 집계
- 건물 마커 클릭 시 요약 정보 팝업
- **객관적 지표 오버레이**:
  - 🚗 주요 차도 (교통량 정보)
  - 🏗️ 공사 현장 (반경 500m 이내)
  - 🚇 지하철역 (소음원)

### 2.3 객관적 지표 & 소음 예측 점수
**목적**: 리뷰가 없는 건물도 예측 가능하도록 객관적 데이터 제공

**데이터 소스**:
- 공공데이터포털: 도로교통소음 측정 자료
- 서울시 열린데이터: 공사정보, 소음진동 배출사업장
- 카카오/네이버 지도 API: 차도 거리, 주변 시설

**소음 예측 점수 알고리즘** (간이 버전):
```
예측 점수 = 기본 점수 3.0
  - 차도 거리 < 50m → +1.0
  - 공사 현장 500m 이내 → +0.5
  - 지하철역 200m 이내 → +0.3
  - 상가 밀집 지역 → +0.4
  (5.0이 가장 시끄러움)
```

**표시 방식**:
```
━━━━━━━━━━━━━━━━━━━━━━━━━━━
📊 객관적 지표 분석
━━━━━━━━━━━━━━━━━━━━━━━━━━━
🤖 AI 예측 소음도: ⭐ 3.8/5.0

🚗 차도 거리: 30m (4차선 도로)
🏗️ 주변 공사: 200m 거리 (2025.12 종료 예정)
🏪 상가 밀집도: 높음 (반경 100m 내 15개)
🚇 지하철역: 150m (2호선 역삼역)
```

### 2.4 소음 리뷰 작성
**목적**: 사용자 참여를 통한 데이터 확보

**입력 항목**:
- 주소 검색 (자동완성)
- 건물명/동/호수 (선택)
- 층수
- 소음 별점 (1~5)
- 소음 유형 (다중선택): 층간소음, 외부소음, 반려동물, 아기, 기타
- 주로 발생하는 시간대 (오전/오후/저녁/새벽)
- 상세 설명 (텍스트)
- 사진 첨부 (선택)

**익명성**:
- 기본: 익명 작성
- 선택: 닉네임 표시

### 2.5 커뮤니티 게시판
**목적**: 소음 외 이웃 관련 정보 공유 및 Q&A

**게시판 구성**:
- 🏘️ **지역별 게시판** (시/구/동 단위)
- 💬 **자유게시판**: 일상적인 이웃 이야기
- 🆘 **소음 고민 상담**: 층간소음 해결 팁, 민원 방법 등
- ❓ **Q&A**: 이사 관련 질문

**기능**:
- 게시글 작성/수정/삭제
- 댓글/대댓글
- 좋아요/북마크
- 신고 기능 (악성 게시물 관리)

### 2.6 소음 측정 및 녹음 기능
**목적**: 실시간 소음 데이터 수집 및 개인 증거 자료 확보

**Phase 1: 안전한 데시벨 측정 (MVP 권장)**
- ✅ **실시간 데시벨(dB) 측정**
  - 스마트폰 마이크 활용
  - 시각적 미터 UI (애니메이션)
- ✅ **시간대별 소음 그래프**
  - 오늘/이번 주/이번 달 통계
  - 최고/최저/평균 데시벨 표시
- ✅ **마이페이지 전용 저장**
  - 개인 증거 수집 목적
  - 커뮤니티 공유 불가
- ✅ **익명 통계만 지도 공유**
  - 데시벨 수치만 전송
  - 위치 범위화 (100m 이상)

**Phase 2: 고급 분석 기능 (Beta)**
- 📊 **주파수 스펙트럼 분석**
  - FFT (Fast Fourier Transform) 활용
  - 저주파(0-250Hz), 중주파(250-3500Hz), 고주파(3500Hz+) 분리
- 🤖 **AI 기반 소음 유형 자동 분류**
  - 충격음 (Impact): 발소리, 의자 끄는 소리
  - 진동음 (Vibration): 세탁기, 청소기
  - 지속음 (Continuous): 차량, 공사 소음
- 📄 **법적 증거용 리포트 생성**
  - 기간별 소음 통계 PDF
  - "이 리포트는 법률 자문이 아닙니다" 명시

**Phase 3: 녹음 기능 (신중한 접근, 조건부 구현)**
- 🎙️ **소음 녹음 (로컬 저장 위주)**
  - 원본 파일은 기기 내부 저장만
  - 암호화된 클라우드 백업 (AES-256)
- 🔇 **음성 주파수 필터링**
  - 대화 음역(300-3400Hz) 자동 제거
  - Band-stop filter 적용
- ⚠️ **커뮤니티 공유 절대 불가**
  - 음성 파일 업로드 차단
  - 통계 데이터만 공유 허용

**3단계 노출 설정**:
```
🔒 레벨 1: 완전 비공개
- 본인만 열람
- 원본 녹음 파일 포함
- 법적 증거 수집 목적

📊 레벨 2: 익명 통계 공유
- 데시벨 수치, 시간대, 소음 유형만
- 위치: 동/면 단위로 범위화
- 원본 녹음 비공개

🌍 레벨 3: 커뮤니티 공개
- 다른 사용자가 볼 수 있는 통계
- 지도 히트맵에 반영
- 음성 파일은 절대 공유 불가
```

**법적 주의사항**:
- 본인 거주지 내 소음 녹음은 합법 (통신비밀보호법 적용 대상 아님)
- 타인의 대화가 명확히 들리는 녹음은 업로드 금지
- 녹음 전 주의사항 팝업 매번 표시
- "법적 증거 수집 목적만 사용" 동의 필수
- 데시벨 측정을 우선 권장, 녹음은 선택 기능

## 3. 기술 스택 (제안)

### 3.1 Frontend
- **프레임워크**: React + TypeScript
- **상태관리**: React Query (서버 상태), Zustand/Recoil (클라이언트 상태)
- **스타일링**: Tailwind CSS
- **지도**: 카카오맵 API 또는 네이버맵 API
- **차트**: Chart.js 또는 Recharts (소음 별점 시각화)

### 3.2 Backend
- **언어/프레임워크**:
  - Option 1: Node.js + Express + TypeScript
  - Option 2: Python + FastAPI
- **데이터베이스**:
  - PostgreSQL + PostGIS (지리 데이터 처리)
- **ORM**: Prisma (Node.js) 또는 SQLAlchemy (Python)
- **인증**: JWT + OAuth (소셜 로그인)

### 3.3 외부 API
- 카카오 주소 검색 API
- 카카오맵 또는 네이버맵 API
- 공공데이터포털 API
  - 도로교통소음 측정 자료
  - 건설공사 정보
  - 소음진동 배출사업장 정보

### 3.4 배포
- **호스팅**: Vercel/Netlify (Frontend), AWS/GCP (Backend)
- **DB**: AWS RDS 또는 Supabase

### 3.5 오디오 처리 기술
- **프론트엔드 오디오 API**:
  - Web Audio API (브라우저)
  - AudioKit / AVAudioEngine (iOS)
  - Android MediaRecorder API
- **데시벨 측정**:
  - 스마트폰 마이크 입력 분석
  - 캘리브레이션 (기기별 보정)
- **주파수 분석**:
  - FFT (Fast Fourier Transform)
  - 실시간 스펙트럼 분석
- **필터링 기술**:
  - Band-stop filter (300-3400Hz 제거)
  - High-pass filter (3500Hz 이상)
  - Low-pass filter (250Hz 이하)
- **데이터 보안**:
  - AES-256 암호화 (로컬 저장)
  - TLS 1.3 (서버 전송)
  - E2E 암호화 (클라우드 백업)

**벤치마킹 모델**:
- **NoiseCapture** (프랑스): 과학 연구용 소음 측정, 데시벨만 수집
- **The Noise App** (영국): 법원에서 증거로 인정받는 측정 앱
- **Alertify** (스마트홈): 프라이버시 우선 설계, 음성/영상 녹음 없음


### 4.2 NoiseReview (소음 리뷰)
```typescript
interface NoiseReview {
  id: string;
  buildingId: string;
  userId?: string; // nullable (익명 가능)
  floor?: number;
  noiseScore: number; // 1-5
  noiseTypes: NoiseType[]; // ['INTER_FLOOR', 'EXTERNAL', 'PET', 'BABY', 'ETC']
  timeOfDay: TimeSlot[]; // ['MORNING', 'AFTERNOON', 'EVENING', 'NIGHT']
  description: string;
  imageUrls?: string[];
  createdAt: Date;
  updatedAt: Date;
}

enum NoiseType {
  INTER_FLOOR = 'INTER_FLOOR',    // 층간소음
  EXTERNAL = 'EXTERNAL',            // 외부소음
  PET = 'PET',                      // 반려동물
  BABY = 'BABY',                    // 아기
  ELEVATOR = 'ELEVATOR',            // 엘리베이터
  TRASH = 'TRASH',                  // 쓰레기 수거
  ETC = 'ETC'                       // 기타
}

enum TimeSlot {
  MORNING = 'MORNING',     // 06-12시
  AFTERNOON = 'AFTERNOON', // 12-18시
  EVENING = 'EVENING',     // 18-24시
  NIGHT = 'NIGHT'          // 00-06시
}
```

### 4.5 NoiseMeasurement (소음 측정 데이터)
```typescript
interface NoiseMeasurement {
  id: string;
  userId: string;
  buildingId?: string;
  timestamp: Date;
  decibelLevel: number; // 데시벨 수치 (dB)
  frequencySpectrum?: FrequencyData; // 주파수 분석 데이터
  noiseType?: NoiseClassification; // AI 분류 결과
  duration: number; // 측정 시간 (초)
  isPublic: boolean; // 공개 여부
  privacyLevel: 'PRIVATE' | 'ANONYMOUS' | 'PUBLIC';
  createdAt: Date;
}

// 주파수 데이터 (음성 복원 불가능)
interface FrequencyData {
  lowFreq: number;   // 0-250Hz (진동, 발소리)
  midFreq: number;   // 250-3500Hz (생활 소음)
  highFreq: number;  // 3500Hz+ (충격음)
  dominantFreq: number; // 주파수 피크
}

enum NoiseClassification {
  IMPACT = 'IMPACT',           // 충격음 (발소리, 의자)
  VIBRATION = 'VIBRATION',     // 진동음 (세탁기, 청소기)
  CONTINUOUS = 'CONTINUOUS',   // 지속음 (차량, 공사)
  MIXED = 'MIXED'              // 복합
}

// 녹음 파일 메타데이터 (파일 자체는 로컬 저장)
interface NoiseRecording {
  id: string;
  userId: string;
  measurementId: string; // NoiseMeasurement 연결
  localFilePath: string; // 기기 내부 경로
  cloudBackupUrl?: string; // 암호화된 백업 URL
  isFiltered: boolean; // 음성 필터링 여부
  fileSize: number; // 바이트
  createdAt: Date;
  // 주의: 이 파일은 절대 서버에 직접 업로드하지 않음
}
```

## 5. 주요 화면 구성

### 5.1 메인 화면
- 상단: 로고 + 검색창
- 중앙: 지도 (히트맵 + 객관적 지표 오버레이)

## 8. 리스크 & 대응 방안

### 8.1 리스크
1. **데이터 부족**: 초기 리뷰가 없으면 가치 하락
   - **대응**: 시드 데이터 확보 (크롤링, 설문), 예측 점수 활용
2. **악성 리뷰**: 허위 정보, 악의적 비방
   - **대응**: 신고 시스템, 모더레이션, 리뷰 신뢰도 점수
3. **개인정보 이슈**: 층/호수 노출로 개인 특정 가능
   - **대응**: 층수는 범위로 표시 (예: 8층 → 6~10층), 호수는 선택 입력
4. **법적 문제**: 부동산 가치 하락 소송
   - **대응**: 이용약관 명시, 사실 기반 리뷰 권장, 면책 조항
5. **녹음 파일 업로드 관련 법적 리스크**: 소음 녹음본에 타인의 대화/음성이 포함될 경우 통신비밀보호법(제3조) 및 개인정보보호법 위반 소지
   - **대응**:
     - 순수 환경 소음(차량, 공사, 발소리 등)만 담긴 녹음은 업로드 허용
     - 타인의 대화·음성이 포함된 녹음은 **업로드 금지** (업로드 시 체크박스 동의 필수)
     - 동/호수 등 개인 특정 가능한 정보 마스킹 가이드라인 제공
     - 업로드 전 면책 동의 (이용약관) 필수
     - 신고 접수 시 즉시 블라인드 처리
     - 녹음 파일보다 **데시벨 측정 수치** 업로드를 우선 권장

### 8.2 KPI (출시 후 모니터링)
- 월간 활성 사용자(MAU)
- 리뷰 작성 수 / 건물당 평균 리뷰 수
- 검색 → 상세 페이지 전환율
- 커뮤니티 참여율 (게시글/댓글 수)
- **소음 측정 기능 사용률**
- **데시벨 데이터 수집량 (지도 품질 지표)**

### 8.3 법적 컴플라이언스 및 안전장치

#### 8.3.1 관련 법률 및 적용 범위

**통신비밀보호법**:
- **제3조, 제14조**: 타인간의 대화 녹음 금지
- **적용 예외**: 환경 소음(차량, 공사, 발소리)은 "대화"가 아니므로 적용 대상 아님
- **판례**: 대법원 2023도8603 - 소음 증거 수집 목적 녹음은 사회상규에 위배되지 않음
- **결론**: 본인 거주지 내에서 소음 녹음은 합법

**개인정보보호법**:
- **제15조, 제23조**: 음성정보는 생체정보(민감정보 가능)
- **적용**: 대화 내용이 포함된 녹음은 개인정보에 해당
- **대응**: 음성 필터링 기술 적용, 커뮤니티 공유는 통계만

**정보통신망법**:
- **제44조의2**: 권리 침해 정보 삭제 및 임시조치 의무
- **플랫폼 책임**: 신고 접수 시 24시간 내 임시조치, 7일 내 최종 결정
- **면책 조건**: 적절한 조치를 취하면 배상책임 경감/면제

#### 8.3.2 필수 이용약관 조항

**제1조 (목적)**
```
본 서비스는 사용자가 거주 환경의 소음 수준을 측정하고 기록하여
개인적 증거 자료로 활용하는 것을 목적으로 합니다.
본 서비스는 법률 자문 서비스가 아니며, 수집된 데이터의 법적 효력을
보증하지 않습니다.
```

**제2조 (사용자 의무)**
```
1. 사용자는 타인의 사생활을 침해하거나 명예를 훼손하는
   목적으로 본 서비스를 사용할 수 없습니다.
2. 사용자는 대화 내용이 명확히 들리는 녹음 파일을
   커뮤니티에 공유해서는 안 됩니다.
3. 사용자는 본인이 업로드한 콘텐츠에 대한 법적 책임을 부담합니다.
```

**제3조 (금지 행위)**
```
다음 행위는 엄격히 금지됩니다:
- 타인의 사적 대화를 의도적으로 녹음하여 업로드하는 행위
- 특정인을 식별할 수 있는 정보와 함께 소음 녹음을 공개하는 행위
- 허위 정보 또는 조작된 소음 데이터 업로드
- 타인을 비방하거나 명예를 훼손하는 게시물 작성
```

**제4조 (플랫폼 권한 및 책임)**
```
1. 회사는 위법하거나 타인의 권리를 침해하는 콘텐츠를
   사전 통지 없이 삭제하거나 임시조치할 수 있습니다.
2. 회사는 사용자가 업로드한 콘텐츠의 적법성을 보증하지 않으며,
   해당 콘텐츠로 인한 법적 분쟁은 사용자의 책임입니다.
3. 회사는 정보통신망법에 따라 권리 침해 신고에 신속히 대응합니다.
```

**제5조 (면책 조항)**
```
1. 회사는 사용자가 본 서비스를 통해 수집한 데이터의
   법적 증거 능력을 보장하지 않습니다.
2. 회사는 사용자 간의 분쟁에 개입하지 않으며,
   법적 책임을 부담하지 않습니다.
3. 회사는 기술적 한계로 인해 모든 위법 콘텐츠를
   사전 차단할 수 없으며, 신고 시 조치를 취합니다.
```

#### 8.3.3 기술적 안전장치

**데이터 수집 단계**:
- ✅ **우선 권장**: 데시벨 수치만 수집 (개인정보 해당 없음)
- ⚠️ **선택 기능**: 주파수 스펙트럼 (음성 복원 불가능)
- ⛔ **신중 접근**: 녹음 파일 (로컬 저장만, 서버 업로드 차단)

**음성 필터링 기술**:
- Band-stop filter: 300-3400Hz 음역 자동 제거
- High-pass filter: 3500Hz 이상 통과 (충격음)
- Low-pass filter: 250Hz 이하 통과 (진동음)

**위치 정보 익명화**:
- 커뮤니티 공유 시 최소 100m 단위로 범위화
- 구체적 주소 대신 "동/면" 단위 표시
- 지도 히트맵: 집계 데이터만 표시

**암호화 및 보안**:
- 로컬 저장: AES-256 암호화
- 서버 전송: TLS 1.3
- 클라우드 백업: E2E 암호화

#### 8.3.4 운영 체계

**신고/삭제 시스템**:
```
1. 신고 접수: 앱 내 신고 버튼 또는 이메일
2. 필수 제출 정보:
   - 신고자 신원 확인 자료
   - 침해 콘텐츠 URL 또는 식별 정보
   - 권리 침해 소명 자료
3. 처리 기간:
   - 접수 후 24시간 이내 임시조치
   - 7일 이내 최종 결정
4. 이의 제기: 조치에 불복 시 14일 이내 재심 요청 가능
```

**법률 자문 체계**:
- 서비스 출시 전 IT/개인정보 전문 변호사 자문
- 이용약관 및 개인정보처리방침 법률 검토
- 분기별 법률 컴플라이언스 점검
- 필요 시 플랫폼 책임보험 가입

**사용자 교육**:
- 녹음 기능 최초 사용 시 주의사항 필수 확인
- "타인 대화 녹음 금지" 매번 팝업 표시
- FAQ에 법적 주의사항 명시
- 증거 리포트에 "법률 자문 아님" 고지

#### 8.3.5 권장 구현 전략

**단계별 리스크 관리**:

| 기능 | 법적 안전성 | 구현 우선순위 |
|------|-----------|-------------|
| 데시벨 측정 | ⭐⭐⭐⭐⭐ 매우 안전 | ✅ Phase 1 (MVP) |
| 주파수 분석 | ⭐⭐⭐⭐ 안전 | ✅ Phase 2 (Beta) |
| 통계 데이터 공유 | ⭐⭐⭐⭐ 안전 | ✅ Phase 2 |
| 녹음 (로컬 저장) | ⭐⭐⭐ 보통 | ⚠️ Phase 3 (조건부) |
| 녹음 파일 공유 | ⭐ 매우 위험 | ⛔ 절대 불가 |

**최종 권장안**:
```
✅ 추천: NoiseCapture 방식
- 데시벨 + 주파수 분석
- 개인정보 리스크 없음
- 과학적 가치 높음

⚠️ 선택: 개인 증거 수집용 녹음
- 마이페이지 전용
- 공유 기능 없음
- 법률 자문 명시

⛔ 금지: 음성 파일 공개
- 법적 리스크 매우 높음
- 플랫폼 존속 위협
```

## 9. 참고 자료

### 9.1 유사 서비스 벤치마킹

**국내 부동산 플랫폼**:
- **직방/다방**: 부동산 리뷰 (소음은 일부 언급만)
- **네이버 부동산**: 단지 정보 (소음 정보 없음)
- **아파트투유**: 아파트 정보 (소음 리뷰 부재)

**해외 부동산**:
- **Zillow** (미국): 부동산 정보, 소음 정보 제한적
- **RentBerry** (유럽): 임대 플랫폼, 소음 데이터 없음

**소음 측정 전문 서비스**:
- **NoiseCapture** (프랑스):
  - Université Gustave Eiffel, CNRS 운영
  - 크라우드소싱 소음 지도
  - 데시벨 수치만 수집 (음성 녹음 없음)
  - 익명 데이터, 오픈 데이터 (ODbL 라이선스)
  - **우리 서비스의 핵심 벤치마킹 대상**

- **The Noise App** (영국):
  - RHE Global 개발
  - 영국 법원에서 증거로 인정
  - 법률 전문가 협력
  - 법적 소송 및 중재 증거 수집 목적

- **Alertify** (스마트홈):
  - 호텔/임대주택용 소음 감지
  - 음성/영상 녹음 없음 (프라이버시 우선)
  - 소음 수준만 모니터링
